## Project Overview
This project includes notebooks for the following Kaggle competition

Predict 5,000 stocks: Will the Close price rise or fall in 30 days?
https://www.kaggle.com/competitions/predicting-stock-trends-rise-or-fall

## Project Structure

* `prediction_random_forest.ipynb`: Contains code and notebooks related to random forest modeling.
* `prediction_regression.ipynb`: Contains code and notebooks related to XGBRegressor modeling.
* `prediction_sarimax.ipynb`: Contains code and notebooks related to SARIMAX modeling.
* `prediction_xgb.ipynb`: Contains code and notebooks related to XGBoost modeling.

## Requirements

* Python 3.11.11
* pandas
* scikit-learn
* statsmodels
* xgboost
